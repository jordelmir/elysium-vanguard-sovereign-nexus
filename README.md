## ⚡ Core Architecture

The system is built on a dual-layer architecture designed for maximum efficiency and security:

1. **Sovereign Engine (Go)**: A high-concurrency backend that interfaces directly with devices via ADB/Fastboot. It handles telemetry streaming, terminal multiplexing, and secure audits.
2. **Nexus Dashboard (JS/HTML/CSS)**: An elite, volumetric UI featuring glassmorphism, real-time data visualization, and a integrated command-line interface.

## 🛠 Features

- **Real-Time Telemetry**: Live tracking of CPU load, RAM usage, Battery health, and Storage metrics.
- **Advanced Terminal**: A persistent, privileged shell environment with automatic context management.
- **Sovereign Auditor**: automated security checks (SELinux status, System Integrity, Root detection).
- **Module Vanguard**: Integrated management and verification of system modules.
- **Performance Hacking**: One-click optimization scripts for A10 and V2 architectures.
- **Sovereign File Manager**: Full filesystem navigation with secure copy and extraction capabilities.

## 🚀 Getting Started

### Prerequisites

- **Go**: v1.21 or higher.
- **Node.js**: For dependency management (dashboard assets).
- **ADB & Fastboot**: Correctly configured in your system PATH.

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/elysium-vanguard.git
   cd elysium-vanguard
   ```

2. Initialize the backend:

   ```bash
   go mod download
   ```

3. Start the Sovereign Engine:

   ```bash
   go run main.go [port]
   ```

   *Note: Defaults to port 3001 if none provided.*

4. Access the Nexus Dashboard:
   Open `http://localhost:3001` in any modern browser.

## 🛡 Security & Privacy

Elysium Vanguard is designed with a "Security First" mindset. No data ever leaves your local environment. All telemetry is streamed over local WebSockets, and terminal sessions are isolated per connection.
